# CAPI Fastly Purger

AWS Lambda handler for CAPI Fastly cache purging.

## Setup

Install dependencies:

```bash
npm install
```

## Build

Compile TypeScript to JavaScript:

```bash
npm run build
```

## Project Structure

```
capi-fastly-purger/
├── src/
│   └── index.ts      # Lambda handler
├── dist/             # Compiled output (generated)
├── package.json
├── tsconfig.json
└── README.md
```
